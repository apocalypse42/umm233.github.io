sort函数小记

list.sort(cmp=None, key=None, reverse=False)
l cmp – 可选参数, 如果指定了该参数会使用该参数的方法进行排序。
l key – 主要是用来进行比较的元素，只有一个参数，具体的函数的参数就是取自于可迭代对象中，指定可迭代对象中的一个元素来进行排序。
l* reverse – 排序规则，reverse = True 降序， reverse = False 升序（默认）。
